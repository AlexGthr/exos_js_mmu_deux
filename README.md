<h1>Exercice 2</h1>

<p><strong>Créer une page web affichant 4 carrés de 200 x 200 pixels avec un fond vert adoptant 
la disposition suivante (voir capture). Ils seront numérotés de 1 à 4.</strong></p>

<p>En cliquant sur n'importe quel carré, celui-ci rétrécit de 10%, décrit une rotation de 360 degrés et 
prend un fond rouge. En recliquant dessus, le carré reprend ses propriétés initiales.</p>
